# FUCINE-MINERVA-2026
Fucine Minerva è un festival multidisciplinare che prende vita negli spazi del Tempio di Minerva Medica di Montefoscoli, luogo simbolico dedicato all’arte, all’artigianato e alla sapienza. Tra musica, poesia, arti visive e artigianato, il festival crea un dialogo tra il valore storico del tempio e le forme espressive contemporanee.
